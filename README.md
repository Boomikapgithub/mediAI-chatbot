# mediAI-chatbot
MediAI is a FastAPI-based platform that allows healthcare consultants to securely register, log in, and share public posts with bios, images, or videos. The system provides a smooth, responsive interface for showcasing consultant expertise and enabling AI-enhanced Responses to user.
languages and tools
Component	    Technology
Frontend	 -   HTML, CSS, Bootstrap
Backend	   -   FastAPI (Python)
Database	 -  SQLite (or any preferred)
Templating	- Jinja2
Storage	  -   Static uploads (images/videos)
Server	  -   Uvicorn
